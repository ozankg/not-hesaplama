# Not Hesaplama Sistemi

## Proje Amacı

Bu proje, öğrencilerin vize ve final notlarını girerek dönem sonu ortalamalarını hesaplamalarını sağlayan basit bir web uygulamasıdır. Kullanıcı tarafından girilen notlar JavaScript kullanılarak hesaplanır ve öğrencinin geçip geçmediği ekranda gösterilir.

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript

## Kurulum Adımları

1. Proje dosyalarını bilgisayara indirin.
2. index.html dosyasını bir web tarayıcısında açın.
3. Alternatif olarak Visual Studio Code üzerinde Live Server eklentisi ile çalıştırın.

## Kullanım Talimatları

1. Vize notunu girin.
2. Final notunu girin.
3. "Hesapla" butonuna tıklayın.
4. Sistem ortalamayı hesaplayarak sonucu gösterecektir.

## Projenin Çalışma Mantığı

Sistem vize notunun %40'ını ve final notunun %60'ını alarak ortalama hesaplar. Ortalama 50 ve üzerindeyse öğrenci başarılı kabul edilir, aksi halde başarısız olarak değerlendirilir.
