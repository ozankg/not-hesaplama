## Proje Adı
Döviz Çevirici

## Projenin Amacı ve Kısa Açıklaması
Bu proje, kullanıcıların farklı para birimleri arasında dönüşüm yapabilmesini sağlamak amacıyla geliştirilmiştir. Kullanıcı miktar girerek Türk Lirası, Dolar ve Euro arasında dönüşüm işlemi yapabilmektedir. Proje basit ve kullanıcı dostu bir arayüze sahiptir.

## Kurulum Adımları

1. Proje dosyalarını bilgisayara indirin.
2. Proje klasörünü Visual Studio Code ile açın.
3. `index.html` dosyasını Live Server ile çalıştırın.

## Çalıştırma / Kullanım Talimatları

1. Miktar alanına dönüştürmek istediğiniz sayıyı girin.
2. İlk kutudan kaynak para birimini seçin.
3. İkinci kutudan hedef para birimini seçin.
4. "Çevir" butonuna basın.
5. Sonuç ekranda görüntülenecektir.
